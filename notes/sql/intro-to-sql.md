




### Relational Database
database is a collection of data managed by software called database management system : software for database management
Type of database : 

 is organized into tables with rows and columns. Each row is a record, and each column is a field or attribute of particular record

- Holds structured data : 

 A database is usually controlled by a database management system (DBMS)



Relationships: Tables are linked together by relationships, which define how the data in one table is related to the data in another table.
Data integrity: The database enforces data integrity rules to ensure that the data is accurate and consistent.
Query language: A query language, such as SQL, is used to retrieve data from the database.


The tables are then linked together by relationships, which define how the data in one table is related to the data in another table. This allows you to efficiently query and retrieve data from the database.
